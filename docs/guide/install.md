# Install Shifu
## Install/Uninstall
### via GitHub repository
1. Clone the Shifu repo via:
   ```
   git clone https://github.com/Edgenesis/shifu.git
   ```
2. Install via:
   ```
   cd shifu
   bash ./test/scripts/deviceshifu-install.sh apply
   ```

To uninstall, issue:
```
bash ./test/scripts/deviceshifu-install.sh delete
```
